# 💑 Gestión de Gastos

Tracker de gastos para parejas e individuales. Built with Next.js + Supabase.

## Stack
- **Frontend**: Next.js 14 + TypeScript + Tailwind
- **Backend**: Supabase (Auth + PostgreSQL + Realtime)
- **Hosting**: Vercel (free)

## Setup

1. Crea un proyecto en [supabase.com](https://supabase.com)
2. Corre el SQL de `/supabase/schema.sql`
3. Copia `.env.example` a `.env.local` y agrega tus keys
4. `npm install && npm run dev`

## Deploy
Conecta el repo a Vercel y agrega las env vars.
